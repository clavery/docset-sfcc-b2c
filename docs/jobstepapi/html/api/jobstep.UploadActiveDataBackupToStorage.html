Overview
Job Step UploadActiveDataBackupToStorage
Upload current site's active data to storage system (as a backup).
This step uploads active data as a backup to storage system during a site move if active data is required to migrate to the new site.
Execution Context:
Sites
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Exit Status
OK
Used when the migration finished successfully.
ERROR
Used when the migration failed. Check the dedicated job log file for the affected job execution for details.
