Job Step IncludeStepsFromJob
Includes steps from another job.
Includes steps from another job. This step makes it easy to create a set of standard steps that you run in multiple jobs.
Execution Context:
Organization or Sites
Parallel Execution:
Supported
Always execute on restart:
Not Supported
Input Parameters
JobID  :  String (Required)
The ID of the job to import steps from.
Exit Status
OK
Step execution finished successfully.
ERROR
Step execution failed.
