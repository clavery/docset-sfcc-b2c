Overview
Job Step ExecuteCategorizationRules
Executes the Categorization based on the Categorization Rules and Conditions defined on a CatalogCategory.
Execution Context:
Organization
Parallel Execution:
Supported
Always execute on restart:
Supported
Input Parameters
CatalogID  :  String (Required)
The Catalog Id which needs Categorization.
CategoryID  :  String (Required)
The Category Id which needs Categorization. If we need to run the Categorization for all Categories, this should be set as 'ALL'.
Exit Status
OK
Step exit status used when the categorization finished successfully.
ERROR
Step exit status used when the categorization failed.
