Properties
attributeID  :  String  (Read Only)
The ID of the product attribute defintion related to this variation attribute. This ID matches the value returned by ObjectAttributeDefinition.getID() for the appropriate product attribute definition. This ID is generally different than the ID returned by getID().
displayName  :  String  (Read Only)
The display name for the product variation attribute, which can be used in the user interface.
