Pipelet PlaceOrder

This pipelet places an order after payment has been authorized. The specified order must be in status CREATED, and will be set to status NEW.

If the order contains product or gift certificate line items associated with product list items, the pipelet updates the purchase the product list items. For example, if the basket contains an item added from a gift registry, the purchase history of the respective gift registry item is updated.

The order will count towards product and customer active data.

The pipelet can be called within or outside of a storefront session, for example in a job pipeline.
