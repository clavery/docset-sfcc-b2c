Pipelet RemoveCustomerAddress
Removes the supplied Address. If the address is associated with a Product List, the address cannot be removed. Under these conditions, the piplet will exit to the error connector.
Group:
Customer
Flags:
Error Connector
Transaction Required
Input Parameters
Address  :  Address (Required)
The address to remove.
Customer  :  Customer (Optional)
DEPRECATED - pipelet parameter will be ignored
