Pipelet GetCategory
The pipelet returns the Category identified by the supplied CategoryID. The pipelet exits with an ERROR if any of the following are true: - a CategoryID is not supplied, - a site catalog cannot be found or - a category with the CategoryID cannot be found in the catalog.
